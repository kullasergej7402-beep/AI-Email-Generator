## Промпт 2 — Регистрация / вход / выход + защита роутов

```
Опираясь на CLAUDE.md и настроенном Supabase, реализуй авторизацию (email + password).

1. Роуты:
   - /signup — форма регистрации (email, пароль, подтверждение пароля), валидация через zod + react-hook-form.
   - /login — форма входа.
   - Кнопка/действие выхода (signOut) в меню пользователя в шапке.
2. Серверные действия (server actions) для signUp / signIn / signOut с корректной обработкой ошибок Supabase: показывай понятные сообщения через toast и инлайн под полем (неверный пароль, пользователь существует, и т.п.). Никаких белых экранов.
3. Middleware: роуты /dashboard, /profile, /pricing/upgrade — только для авторизованных; неавторизованного редиректь на /login. Авторизованного на /login и /signup редиректь на /dashboard.
4. SiteHeader: если пользователь залогинен — показывай avatar + dropdown (Dashboard, Profile, Logout); если нет — кнопки Login / Sign up.
5. После успешного signup/login — редирект на /dashboard.

Покажи, как проверить флоу локально, и убедись, что сессия живёт после перезагрузки.
```
