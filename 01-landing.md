## Промпт 1 — Лендинг

```
Опираясь на CLAUDE.md, собери landing page на роуте "/" (публичный, без авторизации).

Блоки по порядку:
1. Hero — крупный заголовок, подзаголовок про "генерируй профессиональные письма за секунды с помощью AI", две CTA-кнопки ("Попробовать бесплатно" -> /signup, "Узнать больше" -> якорь). Лёгкая анимация появления через framer-motion.
2. Описание сервиса — короткий блок "как это работает" в 3 шага (ввести тему -> выбрать тон и длину -> получить готовое письмо), иконки из lucide-react.
3. Преимущества — сетка из 4-6 карточек (экономия времени, разные тоны, любая длина, история генераций и т.п.).
4. FAQ — аккордеон на 5-6 вопросов (shadcn accordion).
5. CTA-секция перед футером — призыв зарегистрироваться.

Требования:
- Полностью адаптивно: проверь desktop / tablet / mobile, бургер-меню в шапке на мобиле.
- Чистая типографика, аккуратные отступы, не "дефолтный шаблон" — продумай визуальный ритм.
- Все секции — отдельные компоненты в src/components/landing.
- Никаких заглушек-ошибок, ссылки работают.

В конце прогони build и опиши, какие компоненты создал.
```
